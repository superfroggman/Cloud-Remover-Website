C#

Rekommenderas för nybörjare
Cloud Remover C# Edition är den första utgåvan av Cloud Remover. Den kommer med ett fint användargränssnitt och är av robust karaktär. För att använda programmet behöver du en mapp med en serie bilder som input.

Python

Cloud Remover Python Edition är den andra utgåvan av Cloud Remover. Python Edition är en terminalapplikation med möjligheten till många olika videoformat som input.

GitHub Ladda ner

Java

Cloud Remover Java Edition är en snabb terminalapplikation, skriven i det lättanvända språket Java. Programmet har många valmöjligheter och tar emot input i form av en GIF.

GitHub Ladda ner

Android

Cloud Remover Android Edition är en utgåva gjord specifikt för din mobiltelefon. Med denna kan du enkelt spela in en video med mobilen och bli av med moln, smidigare än någonsin.

GitHub Ladda ner

CRrust

CRrust är den senaste och absolut snabbaste versionen av Cloud Remover, skrivet i det avancerade språket RUST. CRrust är även flexibelt och kan utnyttja många olika videoformat.
